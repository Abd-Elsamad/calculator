# 📖 Overview

The Calculator project was built to recreate the experience of a modern digital calculator while focusing on clean code, responsive layouts, and intuitive user interactions.

Beyond performing arithmetic operations, the project demonstrates core frontend development concepts including DOM manipulation, event-driven programming, state management, and user interface design.

---

# ✨ Features

* ➕ Basic Arithmetic Operations
* 🧠 Real-Time Expression Evaluation
* 🗑️ Clear & Delete Functions
* ⌨️ Interactive Button Controls
* ⚡ Fast User Experience
* 📱 Fully Responsive Layout
* 🎨 Minimal & Modern Interface
* 💻 Cross-Browser Compatibility
